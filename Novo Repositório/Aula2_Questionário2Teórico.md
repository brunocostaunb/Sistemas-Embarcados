1. Por que o Linux recebeu esse nome?
Por causa do nome do criador Linus Torvalds.

2. O que são daemons?
Em um sistema operacional, um daemon é um programa de computador que é executado como um processo em segundo plano em vez de estar
sob o controle direto de um usuário interativo.

3. O que é o shell?
Um shell é um interpretador de linha de comando ou que fornece uma interface de usuário tradicional da linha de comando do Unix.
Os usuários direcionam a operação do computador digitando comandos como texto para um interpretador de linha de comando executar,
ou criando scripts de texto de um ou mais desses comandos.

4. Por que é importante evitar executar o terminal como super-usuário?
Porque o super-usuário tem total controle sobre o sistema, podendo manipular arquivos essenciais para seu funcionamento. 
Logo, um usuário inexperiente pode corromper o sistema.

5. Qual botão do teclado completa o que o usuário escreve no terminal, de acordo com o contexto?
A tecla "tab".

6. Quais botões do teclado apresentam instruções escritas anteriormente?
A seta para cima.

7. Apresente os respectivos comandos no terminal para:
  (a) Obter mais informações sobre um comando.
  man [comando]
  
  (b) Apresentar uma lista com os arquivos dentro de uma pasta.
  ls
  
  (c) Apresentar o caminho completo da pasta.
  pwd
  
  (d) Trocar de pasta.
  cd [pasta]
  
  (e) Criar uma pasta.
  mkdir [nome_da_pasta]
  
  (f) Apagar arquivos definitivamente.
  rm [nome_do_arquivo]
  
  (g) Apagar pastas definitivamente.
  rm -r [nome_da_pasta]
  
  (h) Copiar arquivos.
  cp 
  
  (i) Copiar pastas.
  cp -avr [destino1] [destino2]
  
  (j) Mover arquivos.
  mv [nome_do_arquivo] [destino]
  
  (k) Mover pastas.
  mv [pasta1] [pasta1] [destino]
  
  (l) Renomear pastas.
  mv [nome_antigo] [nome_novo]
  
  (m) Apresentar o conteúdo de um arquivo.
  cat [nome_do_arquivo]
  
  (n) Apresentar o tipo de um arquivo.
  file [arquivo]
  
  (o) Limpar a tela do terminal.
  clear
  
  (p) Encontrar ocorrências de palavras-chave em um arquivo-texto.
  grep -rnw [diretorio] -e [texto]
  
  (q) Ordenar informações em um arquivo-texto.
  sort [nome_do_arquivo]
  
  (r) Substituir ocorrências de palavras-chave em um arquivo-texto.
  sed -i [s/original/novo/g] [nome_do_arquivo]
  
  (s) Conferir se dois arquivos são iguais.
  cmp --silent [antigo] [novo] || echo "arquivos sao diferentes"
  
  (t) Escrever algo na tela.
  echo "frase"
